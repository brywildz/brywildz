🛡️ Definition d'un CAPTCHA

Un CAPTCHA (Completely Automated Public Turing test to tell Computers and Humans Apart) est un test variable crée pour les humains visant à differencier les utilisateurs humains et les machines. Ils ont pour but de détourner les robots malveillant des sites web et ainsi prévenir un grand nombre de potentielles attaques tels que :

  • Texte déformé : Un texte très déformé doit être lu et correctement écris pour valider le test.

  • Tests de logique : Ces CAPTCHA sont les plus courants. Ils font passer à l'utilisateur un test de logique simple tel que selectionner les images representants une voiture, réaliser un puzzle et d'autres encore. Ils sont très utilisé car plus ludique pour l'utilisateur.

  • Test Mathematiques : Une equation à résoudre est affiché, l'utilisateur doit la résoudre. Celle-ci est moin fréquente car ils sont très facilement contournable par des robots plus performants.

  • Case "Je ne suis pas un robot" : Une simple case à cocher qui définit via une analyse interne si l'utilisateur est un humain. Nous reviendrons sur son fonctionnement un peu plus tard.

Comme on peut le voir avec ces quelques exemples, il existe une variété de Captcha avec chacun leur qualité et leur defaut. Mais ils ont tous un objectif commun : Limiter les attaques de robots. Nous allons en enumérer quelques unes à présent.

🎯 Utilité des CAPTCHA

Comme dit précedemment les Captcha servent à proteger les sites web des robots maleveillant. Mais de quoi exactement ? En effet les attaques de ces robots sont très variés tout comme il existe une variété de site web. Ceux qui crée ces robots cible des sites avec leurs robots pour des raisons très differents, nous allons vous en présenter quelques unes.

  • Attaques par force brute : un robot tente des milliers de mot de passe sur une page d'authentification afin d'obtenir l'accès à un compte. Le CAPTCHA sert à contrôler qui est en train de s'authentifier, si c'est un robot l'authentification échoue automatiquement.

  • Spam dans les formulaire : Un robot envoie des messages un support fourni par un site web qui sont donc des messages non authentique et peuvent encombrer le site à grand nombre. Le CAPTCHA sert à vérifier si l'auteur du message est humain.

  • DDOS : Un robot envoie plein de requête sur un site dans le but de le surcharger. Le CAPTCHA en contrôlant l'envoie de ces même requête empeche un robot d'en envoyer/

  • Autres : Fraude sur les votes en lignes, Achat de produit exclusif en masse pour de la revente, etc...

Les raisons sont très variés et le Captcha les couvres toutes. C'est pour cela qu'il est extrèmement utilisé sur les sites web. Il offre une solution simple à un problème complexe et très problematiques.
